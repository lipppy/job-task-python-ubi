Timezone information

API call:

  • /timezones
  • /timezones?lat={latitude}&lon={longitude}
  • /timezones?lat={latitude}&lon={longitude}&utccheck=raw

Parameters:

  • without parameter or with unknown parameter(s): displays all available timezones
  • lat: latitude in EPSG:4326 coordinate reference system
  • lon: longitude in EPSG:4326 coordinate reference system
  • utccheck: calculate UTC timezone for "uninhabited" locations or international waters
