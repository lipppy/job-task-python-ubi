Timezone information

API call:

  • {{ request.base_url }}timezones
  • {{ request.base_url }}timezones?lat={latitude}&lon={longitude}
  • {{ request.base_url }}timezones?lat={latitude}&lon={longitude}&utccheck=raw

Parameters:

  • without parameter or with unknown parameter(s): displays all available timezones
  • lat: latitude in EPSG:4326 coordinate reference system
  • lon: longitude in EPSG:4326 coordinate reference system
  • utccheck: calculate UTC timezone for "uninhabited" locations or international waters

Examples of API calls:

  • {{ request.base_url }}timezones?lat=48.19&lon=16.39 - returning timezone "Europe/Vienna"
  • {{ request.base_url }}timezones?lat=47.49&lon=19.04 - Budapest, Hungary - returning timezone "Europe/Budapest"
  • {{ request.base_url }}timezones?lat=34.02&lon=-118.49 - Los Angeles, USA - returning timezone "America/Los_Angeles"
  • {{ request.base_url }}timezones?lat=-81.31&lon=-175.49 - Ross Ice, Antarctica - returning timezone "UTC-12"
  • {{ request.base_url }}timezones?lat=1.75&lon=-157.3 - a little north from (1°) Kiribati, Christmas Island, Pacific Ocean - returning timezone "Pacific/Kiritimati"
  • {{ request.base_url }}timezones?lat=2.75&lon=-157.3 - a little north from (1°) Kiribati, Christmas Island, Pacific Ocean - returning timezone "UTC+14" (from UTC zones data)
  • {{ request.base_url }}timezones?lat=2.75&lon=-157.3 - a little north from (1°) Kiribati, Christmas Island, Pacific Ocean - returning timezone "UTC-10" (by manually calculated UTC zones)
